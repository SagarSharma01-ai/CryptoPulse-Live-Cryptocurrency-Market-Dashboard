## 📌 Project Overview
CryptoPulse is a real-time, highly interactive Power BI dashboard designed to track and analyze the live performance of the top 100 cryptocurrencies. Built with a dark-mode "Cyberpunk/High-Tech" UI, it closely mimics the aesthetic of premium crypto exchange applications (like Binance or CoinDCX).

<img width="1500" height="1000" alt="PhotoshopExtension_Image (2)" src="https://github.com/user-attachments/assets/a030c8f3-951c-49a5-906b-0593d68fd011" />

## 🚀 Key Features & Live Integration
* **Live API Data:** Integrated with the **CoinGecko API** using Power Query to fetch real-time market data without requiring manual Excel dumps.
* **Market Health KPIs:** Live tracking of Global Market Cap, 24H Trading Volume, and flagship asset prices (Bitcoin & Ethereum).
* **Volatility Tracking:** Dynamic Horizontal Bar Charts identifying the Top 5 Gainers (Bullish Movers) within a 24-hour window.
* **Market Dominance:** Visualized the liquidity flow and market share (Bitcoin vs. Altcoins) using custom DAX measures.
* **Pro-Level UI/UX:** Built a completely transparent matrix table featuring conditional formatting (Up/Down indicators) for real-time profit/loss tracking.

## 🛠️ Tools & Techniques Used
* **Tool:** Microsoft Power BI
* **Data Extraction:** Web API Integration (Power Query / M-Code)
* **Data Modeling & Analytics:** Advanced DAX (CALCULATE, SUM, IF, Dynamic Filtering)
* **Design:** Custom Dark Theme, Glassmorphism effects, and Conditional Formatting
